# 🐱 Kai的胖丁 - 像素电子宠物

一只住在你浏览器里的小橘猫，喂它、陪它玩、看它写代码。

**在线体验：** [https://kai2yan.github.io/pixel-cat-pet/](https://kai2yan.github.io/pixel-cat-pet/)

## 玩法

### 🐾 基本互动

| 按钮 | 效果 | 消耗 |
|------|------|------|
| 🐟 喂鱼 | 饱腹 +20，心情 +5 | - |
| 🧶 玩耍 | 心情 +20，亲密 +5 | 精力 -15 |
| 💤 睡觉 | 精力 +30 | 进入睡眠状态 |
| 🤗 撸猫 | 亲密 +10，心情 +8 | - |
| 💻 编程 | 心情 +10，亲密 +3 | 精力 -10 |

每次互动都会获得金币和经验值。

### 📊 四大属性

- **🐟 饱腹** — 会随时间下降，太低猫咪会盯着鱼缸发呆
- **😺 心情** — 多陪它玩就会开心，不理它会难过
- **💤 精力** — 玩耍和编程消耗精力，睡觉恢复
- **❤️ 亲密** — 撸猫、玩耍慢慢积累，解锁更多互动

### 🏪 商店

用金币购买消耗品和配饰：

- **消耗品**：小鱼干、猫薄荷、猫咪咖啡、小蛋糕，直接恢复属性
- **配饰**：皇冠、领结、墨镜、围巾等 8 种装饰，买了可以给猫咪戴上

### ⬆️ 等级系统

通过互动获得经验升级，共 8 个等级：

| 等级 | 称号 |
|------|------|
| Lv.1 | 小奶猫 |
| Lv.2 | 调皮猫 |
| Lv.3 | 好伙伴 |
| Lv.4 | 代码猫 |
| Lv.5 | 黑客猫 |
| Lv.6 | 编程大师 |
| Lv.7 | 猫中之王 |
| Lv.8 | 传说喵神 |

达到 **Lv.8** 会解锁隐藏皮肤「📦 大胖丁の箱子」。

### 🎮 接鱼小游戏

点击「小游戏」进入 30 秒接鱼挑战：

- 用鼠标/触屏控制猫咪左右移动
- 接住掉落的鱼得分（+1 🐟）
- 躲开炸弹（碰到 -5 秒）
- 得分兑换金币（1 鱼 = 2 金币）

### 🏆 成就

共 14 个成就可解锁，包括初次投喂、美食家、代码高手、捕鱼达人等，每个成就奖励 20 金币。

### 🌤️ 其他系统

- **天气**：自动随机切换晴天/下雨/下雪/多云，带动态特效
- **昼夜**：跟随真实时间变化，白天有阳光，夜晚有星星
- **日记**：猫咪会自动记录每天发生的事情
- **连续访问**：每天来看看猫咪，累计连续天数

## 技术

纯前端项目，单个 `index.html` 文件，无需后端：

- 原生 HTML / CSS / JavaScript
- localStorage 本地存储游戏进度
- Web Audio API 音效
- Canvas 小游戏
- 猫咪图片由 Gemini 生成

## 本地运行

直接用浏览器打开 `index.html` 即可，或者：

```bash
# 任意静态服务器
npx serve .
```
